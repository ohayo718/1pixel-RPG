# 1pixel-RPG 🎮

**世界で最もミニマルな RPG** - 全てが 1 ピクセルで表現される想像力を刺激するゲーム体験

## 🎯 概要

1pixel-RPG は、極端な抽象化によって想像力を刺激する実験的な RPG です。マップ、プレイヤー、敵、街まで、全てが 1 ピクセルの色で表現されます。

### ✨ 特徴

- **🟦 1 ピクセル表現** - 全ての要素が 1 ピクセルの色で表現
- **🎧 立体音響** - Web Audio API + PannerNode による空間オーディオ
- **⌨️ キーボード操作** - 全ての操作がキーボードで可能
- **🔄 エンドレスモード** - ボスを倒した後もプレイ継続可能
- **💡 光の表現** - HP に応じて自分の「光」が消えていく演出

## 🎮 遊び方

### 🚀 起動方法

1.Code から Zip をダウンロード
2.Zip を解凍。 3.ブラウザで index.html を開く。

### 操作キー

| キー            | 動作                               |
| --------------- | ---------------------------------- |
| WASD / 矢印キー | 移動                               |
| Space           | インタラクション                   |
| 1-4             | メニュー選択（バトル・街・道具屋） |
| ESC             | ポーズメニュー / キャンセル        |
| M               | ミュート切り替え                   |

### ピクセルの色

| 色          | 意味               |
| ----------- | ------------------ |
| 🟦 シアン   | プレイヤー（自分） |
| 🟧 オレンジ | 街                 |
| 🟥 赤       | 敵                 |
| 🟪 マゼンタ | ボス               |
| 🟨 黄色     | 宝箱               |
| 🟩 緑       | 森                 |
| 🟦 青       | 水                 |

## 🛠️ 技術スタック

- **HTML5 Canvas** - ゲーム描画
- **Web Audio API** - 立体音響（HRTF、PannerNode）
- **Vanilla JavaScript** - フレームワーク不使用
- **CSS3** - UI/アニメーション
